üîç API Endpoints

Limited Edition Drops API

  • GET /limited-drops - Get all drops
  • POST /limited-drops - Create new drop
  • PUT /limited-drops/:id - Update drop
  • DELETE /limited-drops/:id - Delete drop
  • GET /limited-drops/:id/products - Get drop products
  • POST /limited-drops/:id/products - Add products to drop
  • DELETE /limited-drops/:id/products/:productId - Remove product
  • GET /limited-drops/active/frontend - Get active drop for frontend
