Buy Now Functionality Examples

Method 1: Using JSON Object (Recommended)

This method passes all product data as a single JSON object:

<button onclick="buyNow({ id: '123', name: 'AJ 1 Low x Travis Scott Fragment', price: 10499, image: 'uploads/images/product1.jpg', size: '9', quantity: 1 })">Buy Now</button>

Method 2: Using Individual Parameters

This method passes each parameter separately:

<button onclick="buyNowSimple('123', 'Product Name', 10499, 'image.jpg', '9', 1)">Buy Now</button>

How to Implement in Your Product Pages

1. Add the buyNow function to your product page:

<script> function buyNow(productData) { const encodedData = encodeURIComponent(JSON.stringify(productData)); window.location.href = `checkout.html?product=${encodedData}`; } function buyNowSimple(id, name, price, image, size, quantity) { const params = new URLSearchParams({ id: id, name: name, price: price, image: image || 'uploads/images/1754636223677-118256129.png', size: size || 'Standard', quantity: quantity || 1 }); window.location.href = `checkout.html?${params.toString()}`; } </script>

2. Use it in your product detail page:

<button onclick="buyNow({ id: '${product.id}', name: '${product.name}', price: ${product.price}, image: '${product.image_url}', size: selectedSize, quantity: selectedQuantity })" class="btn btn-primary">Buy Now</button>

3. Or use the simple version:

<button onclick="buyNowSimple('${product.id}', '${product.name}', ${product.price}, '${product.image_url}', selectedSize, selectedQuantity)" class="btn btn-primary">Buy Now</button>
