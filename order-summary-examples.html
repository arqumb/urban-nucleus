üîó How to Implement in Your System

1. Add the functions to your product pages:

<script> // For direct product checkout function buyNow(productData) { const encodedData = encodeURIComponent(JSON.stringify(productData)); window.location.href = `checkout.html?product=${encodedData}`; } // For simple parameter checkout function buyNowSimple(id, name, price, image, size, quantity) { const params = new URLSearchParams({ id: id, name: name, price: price, image: image || 'uploads/images/1754636223677-118256129.png', size: size || 'Standard', quantity: quantity || 1 }); window.location.href = `checkout.html?${params.toString()}`; } // For cart checkout function checkoutCart(cartId) { window.location.href = `checkout.html?cartId=${cartId}`; } // For order review function reviewOrder(orderId) { window.location.href = `checkout.html?orderId=${orderId}`; } </script>

2. Use in your product detail page:

<!-- Buy Now Button --> <button onclick="buyNow({ id: '${product.id}', name: '${product.name}', price: ${product.price}, image: '${product.image_url}', size: selectedSize, quantity: selectedQuantity })" class="btn btn-primary">Buy Now</button> <!-- Add to Cart Button --> <button onclick="addToCart('${product.id}')" class="btn btn-secondary">Add to Cart</button>

3. Use in your cart page:

<!-- Checkout Button --> <button onclick="checkoutCart('${cartId}')" class="btn btn-success">Proceed to Checkout</button>

4. Use in your order history:

<!-- Review Order Button --> <button onclick="reviewOrder('${order.id}')" class="btn btn-info">Review Order</button>
